# UCNS object canon — restart boundary (post-Chapter-1)

## Primitive invariant

A UCNS object is a directed twofold branched angular cover carrier.
Null is the unique complete absence of distinction.
Zero (Structural Null) is not a seam that can be chosen, moved, or normalized away; it is the single point where the otherwise doubled carrier is intrinsically single.

## Immediate obligations (from sealed Chapter 1)

A conforming object definition must make the following unrepresentable:

1. an object with \( B = 0 \) that is not \(\mathbf{N}\)
2. identification of algebraic zero with Structural Null
3. a 360° displacement treated as automatic negation / frame inversion / complete return
4. functional dependence among \( W, M, B \) above null
5. pruning of a cell that retains any structural distinction merely because its payload is algebraically zero
6. collapse to null without complete structural erasure
7. additive real breadth that is also faithful (the no-go proposition)

No multiplication, factorization, embedding, codec, or theorem is promoted into the new root until it is defined over this object and preserves these obligations.
