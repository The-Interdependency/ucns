# UCNS object canon — restart boundary

## Primitive invariant

A UCNS object is a recursive Möbius carrier. Its unique intrinsic twist is
the seam at which zero exists. Zero is hidden by topology: it is not one of
the object's observable nonzero positions and is not chosen by coordinates.

## Immediate obligations

A replacement object definition must make the following unrepresentable:

1. a `UCNSObject` with no intrinsic twist;
2. normalization that chooses an object's origin from its first anchor;
3. identification of geometric zero with the multiplicative identity;
4. face state stored without a lawful relation to seam-crossing parity;
5. a recursive payload that is not itself twist-bearing;
6. a transformation that moves or deletes the seam;
7. a 360-degree traversal treated as complete return.

No multiplication, factorization, embedding, codec, or theorem is promoted
into the new root until it is defined over this object and preserves these
obligations.
